# Bookmark Shelf for Brave

Brave 向けのマウス中心ブックマークランチャー兼整理 UI です。

## できること

- 拡張アイコン、ショートカット、右クリックメニューからサイドパネルを表示
- よく使うブックマークを最上部に表示
- よく使うもの、ジャンル欄の折りたたみ表示
- よく使うもののフル画面表示切り替え
- ジャンル別表示と一覧表示を切り替え
- カード表示とコンパクト表示を切り替え
- ダークモード切り替え
- ファビコン、ドメイン、URL、フォルダ名を使ったプレビュー表示
- ブックマーク名、URL、ジャンルの絞り込み
- ドラッグ&ドロップでジャンル変更
- ジャンルの追加、名前変更、削除
- カード内のセレクトからブックマークのジャンルを即変更
- 未分類ブックマークを自動でジャンル割り振り
- スライダーで一覧の表示量を調整
- 画像ファイルを指定して壁紙を変更、カードにも薄く反映
- スライダーで壁紙の透明度を調整
- ブックマーク名、URL、ジャンル、固定状態を編集
- 削除モードでカードを即削除、別アイコンからリンク先を開く
- 表示中のブックマークのリンク切れ候補をチェック
- ルールベースの自動ジャンル割り振り

## Brave で読み込む

1. Brave で `brave://extensions/` を開きます。
2. 右上の「デベロッパーモード」を有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」を押します。
4. このフォルダを選びます。

ショートカットは標準で `Ctrl+Shift+Y` です。必要なら `brave://extensions/shortcuts` から変更できます。
